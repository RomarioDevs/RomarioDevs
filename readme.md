## Jogos e demonstrações

Atualmente o repositório contém (ou planeja conter) pequenos jogos e animações. Cada jogo terá sua própria pasta em `games/` com um arquivo HTML auto-contido para rodar localmente.

- Snake (animação do contribution grid e jogo clássico) — `games/snake.html`
- Clicker — `games/clicker.html`
- Comando/Console Demo — `games/terminal-game.html` (simulação de comandos e respostas)

> Observação: se os arquivos ainda não existirem, serão adicionados em commits futuros. Veja abaixo como rodar localmente.

## Como executar os jogos localmente

1. Clone o repositório:

```bash
git clone https://github.com/RomarioDevs/RomarioDevs.git
```

2. Abra o arquivo HTML do jogo no navegador. Por exemplo:

```bash
# Abrir Snake localmente
xdg-open games/snake.html
```

Ou simplesmente arraste o arquivo HTML para o navegador.

## Como contribuir

- Abra uma issue para sugerir um jogo ou melhoria.
- Faça um fork e envie um pull request com mudanças pequenas e documentadas.

### Convenções de commit

Use mensagens claras. Exemplo:

```
feat: adicionar jogo clicker
fix: corrigir colisão no snake
docs: atualizar README com instruções
```
